# Amprio Milano B2B Landing Page

## Описание проекта

Лендинг-страница для B2B-сегмента Amprio Milano.

## Технологии

Проект построен с использованием:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Установка и запуск

### Требования

- Node.js & npm (или bun) - [установка через nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Шаги для запуска

```sh
# Шаг 1: Установить зависимости
npm install
# или
bun install

# Шаг 2: Запустить dev-сервер
npm run dev
# или
bun dev
```

Проект будет доступен по адресу `http://localhost:8080`

## Скрипты

- `npm run dev` - запуск dev-сервера
- `npm run build` - сборка для production
- `npm run build:dev` - сборка в режиме development
- `npm run lint` - проверка кода линтером
- `npm run preview` - предпросмотр production сборки
